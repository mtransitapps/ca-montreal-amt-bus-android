Title: Bus Express AMT Montréal (MT)
----------
Short description: Informations du bus Express AMT de Montréal pour MT.
----------
Full description:
BETA : TEST EN COURS
Pour quitter le test : http://goo.gl/c19oif

Cette application ajoute les informations du bus Express AMT de Montréal dans MT.

Une fois cette application installée, l'application MT affichera les informations du bus (horaires ...).

Cette application a seulement une icône temporaire et pas d'écran : télécharger l'app MT (gratuit) dans la section "Autres ..." ci-dessous ou en cliquant sur ce lien Google Play http://goo.gl/pCk5m

Vous pouvez installer cette application sur la carte SD mais ce n'est pas recommandé.

Ces horaires sont générées à partir du fichier GTFS fourni par l'Agence Métropolitaine de Transport.
http://www.amt.qc.ca/developpeurs/

Cette application est gratuite et open-source : 
https://github.com/mtransitapps/ca-montreal-amt-bus-android

Cette application n'est pas associée à l'Agence Métropolitaine de Transport.
